# 📝 更新日志

## v1.0.0 (2026-06-12)

* 首次发布。
* 新增 网盘资源搜索功能，对接 PanSou 搜索引擎。
* 新增 搜索结果分页浏览，支持 next/n/下一页、prev/p/上一页 翻页。
* 新增 单条结果详情查看，支持图片展示与链接有效性标记。
* 新增 按盘类型过滤结果，支持中英文别名（如 bd/百度、kk/夸克）。
* 新增 链接有效性自动检测，列表检前 5 条、详情检前 10 条。
* 新增 自动验链开关，开启时隐藏失效链接，关闭时仅标注不隐藏。
* 新增 智能条目过滤，所有链接检测且全部失效时自动隐藏条目。
* 新增 全局排除关键词配置。
* 新增 多搜索来源切换（全部 / 仅 Telegram / 仅插件）。
* 新增 JWT 认证支持，自动续期 Token。
* 新增 HTTP 代理配置支持。
* 新增 子命令 help/帮助 查看帮助信息。
